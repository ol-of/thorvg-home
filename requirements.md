# 홈페이지 요구사항 명세

## 목적
- ThorVG 홈페이지의 기본 구조와 스타일 시스템을 정의한다.

## 핵심 요구사항
1. 다크/라이트 테마를 지원한다.
2. 모바일 환경을 지원하는 반응형 레이아웃을 제공한다.
3. 스타일은 별도의 CSS 파일로 분리한다.
4. 사이트는 다음 페이지 구성을 기준으로 한다.
   - Home
   - About
   - Showcase
   - Docs
   - Blog
   - Tools

## 테마 정책
- 다크 테마 배경색은 `#212121`을 사용한다.
- 라이트 테마에서 주요 컨텐츠 텍스트 색상은 `#121212`를 사용한다.
- 기본 HTML/CSS 구조는 두 테마를 모두 수용할 수 있어야 한다.

## 레이아웃 정책
- 모바일 우선 방식으로 작성한다.
- 내비게이션, 히어로 섹션, 페이지 링크 영역을 포함한 기본 랜딩 페이지를 제공한다.
- 추후 각 페이지 확장을 고려해 공통 헤더/메인/푸터 구조를 유지한다.

## 산출물
- `index.html`
- `styles.css`
- 본 요구사항 문서(`requirements.md`)
